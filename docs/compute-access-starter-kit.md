# Compute Access Starter Kit

> Public starter kit for founders, SMEs, researchers, public-sector users, and open-source teams evaluating European compute access. Last checked: 2026-06-17.

Limes Labs is organizing public knowledge about legitimate access paths. Limes Labs does not claim EuroHPC, AI Factory, or IT4LIA compute access, partnership, contact progress, or eligibility unless a public-safe record is logged.

## Minimum Artifact Checklist

Prepare these before sending a helpdesk question or portal application:

- Legal entity or host partner: organisation name, country, VAT number if applicable, PIC if available, PI/contact, and affiliation.
- Project one-pager: problem, applicant, target users, expected outputs, public value, and why EuroHPC or IT4LIA is needed.
- Access route hypothesis: Playground, Fast Lane, Large Scale, AI for Science, Benchmark, Development, Regular, Extreme Scale, or IT4LIA TRY/Medium/Large.
- Compute estimate: target system if known, GPU-hours or node-hours, storage, duration, checkpoint/data movement, and fallback system.
- Benchmark evidence: small run logs, throughput, memory, scaling results, and bottlenecks.
- Data and licence statement: dataset sources, licences, sensitive/personal data status, and publication plan.
- Model card or system card draft: intended use, limitations, evaluations, release terms, and safety notes.
- Safety/compliance plan: civilian purpose, AI Act risk notes, abuse/misuse mitigations, and reporting process.
- Reproducibility package: public repo links, container/environment, run command, dependency licences, and checkpoint/resume plan.

## Source Rule

Every factual program claim in this starter kit should either cite an official source URL with `date_checked` and `status`, or be explicitly marked as Limes interpretation / needs verification. See [source-policy.md](source-policy.md).
